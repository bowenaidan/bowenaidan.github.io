This repository is now set up as a Ruby/Jekyll site for a personal blog + backlog + portfolio.

## Run locally

```bash
bundle install
bundle exec jekyll serve
```

Open `http://127.0.0.1:4000`.

## Content editing

- Blog posts: `_posts/*.md`
- Backlog page: `backlog.md`
- Projects page: `projects.md`
- Global layout: `_layouts/default.html`
- Styling: `assets/css/style.css`

## Build

```bash
bundle exec jekyll build
```
